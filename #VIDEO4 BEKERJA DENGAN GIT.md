# BEKERJA DENGAN GIT

**Git Command (local)**  
1. Inisialisasi repo git di lokal
~~~
$ git init
~~~  
2. Menambah file ke **staging area**
~~~
$ git add <file(s)>
~~~
3. Mengetahui status repo
~~~
$ git status
~~~
4. Melakukan konfigurasi ke dalam git
~~~
$ git config
~~~
5. Membuat branch
~~~
$ git branch
~~~
6. Melihat bantuan command lainnya
~~~
$ git help
